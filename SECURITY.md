# Security Policy

## Supported versions

| Version | Supported |
| ------- | --------- |
| 1.x     | Yes       |

## Reporting a vulnerability

If you discover a security issue in fullstack-swarm (for example unsafe defaults in agent prompts, secret leakage in scaffolded examples, or supply-chain concerns in packaging), please report it privately.

**Email:** mandar.zope@growisto.com

Please include:

- A short description of the issue
- Steps to reproduce (if applicable)
- Impact assessment (what an attacker could do)

Do **not** open a public GitHub issue for security reports until we have had a chance to respond.

We aim to acknowledge reports within 7 days and share a remediation plan or fix timeline as soon as we can assess the issue.
