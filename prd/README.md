# PRD folder

Drop your Product Requirement Document here as `current.md`. The `project-manager` subagent reads from this exact path.

Older PRDs can be archived as `vN.md` (e.g. `v1.md`, `v2.md`) for reference. Only `current.md` is read by the swarm.

If you change `current.md` mid-flight, tell the orchestrator explicitly — it will not re-decompose unless asked.
